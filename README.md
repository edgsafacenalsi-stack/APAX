# APAX

> *Una sola vez. Tuyo para siempre.*

APAX es un asistente de IA personal para Android, construido con React Native + Expo.
Habla en español (con opción de inglés), escucha tu voz, actúa, y muestra cada acción en tiempo real.

## Módulos
- `services/` — Claude API, STT, TTS
- `components/` — HUD, log de acciones, burbuja de voz
- `app/` — pantallas principales
- `store/` — estado global
- `assets/` — fuentes, íconos

## Stack
- React Native + Expo
- Claude API (Anthropic)
- Expo Speech
- Reanimated 2
